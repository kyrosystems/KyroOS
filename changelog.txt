# Changelog

## 26.03.12 | В ПРОЦЕССЕ РАЗРАБОТКИ

### Major Features
- **Driver Manager (Full Implementation):**
    - Реализована полная поддержка Loadable Kernel Modules (LKM).
    - Добавлены команды shell: `lsmod`, `insmod`, `rmmod`.
    - Поддержка динамической загрузки/выгрузки драйверов.
    - Верификация подписи модулей через RSA+SHA256.
    - ELF-релокация и загрузка исполняемого кода.
- **KyroFS (Enhanced Implementation):**
    - Улучшена структура on-disk формата.
    - Добавлена поддержка журналирования.
    - Реализована система экстентов для эффективного хранения файлов.
    - Поддержка монтирования/размонтирования разделов.
    - Inode-based структура с полным набором метаданных.
- **Installer (Full Implementation):**
    - Интерактивный установщик с TUI интерфейсом.
    - Поддержка выбора диска для установки.
    - Форматирование в KyroFS.
    - Автоматическая установка загрузчика Limine.
    - Копирование всех системных файлов.
- **POSIX Commands (40+ New Commands):**
    - Файловые: `cp`, `mv`, `rm`, `cat`, `ls`, `cd`, `pwd`, `mkdir`, `rmdir`, `touch`.
    - Shell: `echo`, `env`, `export`, `unset`, `alias`, `unalias`, `history`, `clear`, `reset`.
    - Процессы: `ps`, `kill`, `killall`, `nice`, `renice`.
    - Утилиты: `sleep`, `date`, `time`, `uptime`.
    - Текст: `head`, `tail`, `wc`, `grep`, `find`, `sort`, `uniq`.
    - Система: `chmod`, `chown`, `ln`, `df`, `du`, `free`, `mount`, `umount`, `fdisk`, `mkfs`.
    - Информация: `uname`, `hostname`, `whoami`, `id`, `groups`, `users`.
    - Питание: `reboot`, `halt`, `poweroff`.
    - Стандартные: `true`, `false`, `test`, `yes`, `sync`.
- **Memory Manager Upgrade:**
    - Улучшенный физический менеджер памяти (PMM).
    - Виртуальная память с demand paging.
    - Поддержка huge pages (2MB, 1GB).
    - Подготовка для поддержки до 100 ПБ RAM.

### Shell Improvements
- **Prompt Colors:** Зелёный цвет символа `>` через ANSI escape-коды.
- **Ctrl+C/D Handling:** Корректная обработка Ctrl+C/D во всех приложениях.
- **3d Demo:** Выход через Ctrl+C/D или Q/ESC.

### Userspace Improvements
- **libc Enhancements:** Добавлены `print_u32`, `print_u64`, `print_hex`.
- **TUI Library:** Улучшена для использования в установщике.
- **Graphics:** Поддержка framebuffer в userspace приложениях.

### System Fixes
- Улучшена стабильность планировщика.
- Оптимизировано использование памяти.
- Исправлены ошибки в VFS.

## 26.03.11 
- **Networking (Extreme Upgrade):**
    - Внедрена поддержка TCP (машина состояний SYN/ACK).
    - Улучшен `wget`: теперь проверяет корректность IP (0-255), поддерживает базовый DNS и парсинг заголовков.
    - `ifconfig` теперь показывает полную статистику интерфейса.
- **Graphics & GUI:**
    - Реализовано Двойное Буферирование (Double Buffering) для устранения мерцания.
    - Добавлен `browser` — графическое приложение для просмотра контента.
    - Добавлена `3dgame` — полноценная мини-игра с 3D перспективой.
- **Shell & System:**
    - Поддержка `Ctrl+C` для прерывания любых длительных процессов (ping, 3d).
    - Восстановлены и исправлены команды `touch`, `mkdir`, `3d`.
    - Системные файлы (документация) теперь вшиты в образ и доступны через `/docs`.
- **Kernel Fixes:**
    - Исправлен лимит TSS в GDT (sizeof entry - 1).
    - Версия ядра синхронизирована с версией ОС.
